Atomic Simulator

Open `index.html` in a browser to run the simulator locally.

Features:
- Home page title "Atomic Simulator" with three levels: Easy, Medium, Hard.
- Interactive atom visualization: nucleus (protons/neutrons) and orbiting electrons.
- Clicking particles opens an information panel with level-specific content.
- Particle colors: Proton (red), Electron (blue), Neutron (gray).

Try it:
1. Open `index.html` in your browser (double-click or serve with a simple HTTP server).
2. Click the level buttons, then click particles to view explanations.

Development:

- CSS: css/styles.css
